# Archivos de Invitaciones de Torneos

Esta carpeta contiene los archivos de invitaciones de los torneos.

## Tipos de archivo permitidos:
- PDF (.pdf)
- Documentos: DOC, DOCX

## Convención de nombres:
- `invitation_{tournament_id}_{timestamp}.{extension}`

## Acceso:
- Solo usuarios autenticados pueden subir archivos
- Los archivos son accesibles públicamente para visualización
