# Archivos de Normas de Torneos

Esta carpeta contiene los archivos de normas y reglamentos de los torneos.

## Tipos de archivo permitidos:
- PDF (.pdf)
- Imágenes: JPG, JPEG, PNG, GIF

## Convención de nombres:
- `norms_{tournament_id}_{timestamp}.{extension}`

## Acceso:
- Solo usuarios autenticados pueden subir archivos
- Los archivos son accesibles públicamente para visualización
