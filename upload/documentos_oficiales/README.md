# Documentos oficiales de dominó

Coloque aquí los archivos PDF (o .doc, .docx) que desee publicar en la sección **Documentos oficiales** del landing.

- Se listan automáticamente en la web.
- Los visitantes pueden **ver en línea** (consulta/lectura en el navegador) o **descargar**.
- Formatos admitidos: PDF, DOC, DOCX.

Nombres de archivo recomendados: sin espacios, por ejemplo `reglamento_fvd.pdf`, `normas_torneos.pdf`.
