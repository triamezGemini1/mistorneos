# Carpeta de Logos

Esta carpeta contiene los logos de los clubs del sistema.

## Formatos permitidos:
- JPG/JPEG
- PNG
- GIF
- PDF

## Tamaño máximo:
- 5MB por archivo

## Estructura de nombres:
- Los archivos se renombran automáticamente como: `logo_[club_id]_[timestamp].[extensión]`
- Ejemplo: `logo_1_1695823456.jpg`

## Seguridad:
- Solo se permiten los formatos especificados
- Los archivos PHP están bloqueados
- Acceso controlado por .htaccess
